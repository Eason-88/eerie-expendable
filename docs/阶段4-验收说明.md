# 阶段 4 验收说明

## 目标

打磨表现与手感：战术人形（非沙包）、可见步枪、雾林氛围、音效与 CS 风格语音播报。

## 客户端启动

```powershell
cd d:\project\heiying\client\web-preview
python -m http.server 5173
```

打开 http://127.0.0.1:5173 （建议 Ctrl+F5）。**首次点击画面**后解锁音频与语音。

美术参考：`assets/view-pseudo-3d.png`

## 检查项

| 项 | 预期 |
|----|------|
| 玩家外观 | 头盔+耳机、战术背心/MOLLE、大腿枪套、手持步枪 |
| 敌人外观 | 同结构异色（敌人为偏褐红；叛变友军为深绿） |
| 掩体 | 风化木箱（木条箍），非纯色方块 |
| 场景 | 浓雾、针叶林、土路、路径旁木箱道具 |
| 射击 | 枪口出弹道；**采样枪声**（CS 风格短脆裂音+回声）+ 后坐力 + 镜头微震 |
| 脚步 | `audio/footstep.wav` 采样 |
| 受击 / 换弹 | 采样音效 |
| 左下角进度 | Edge 神经 TTS（`/api/v1/tts/speak`），失败回退浏览器语音 |
| 左上角对讲机 | 同上 + 对讲机带通滤波 |
| 入场 | 开局进度播报 + 总部呼叫 VO |

## 音频说明

- 枪声等：`client/web-preview/audio/*.wav`（本地合成的 FPS 风格采样，非 Valve 版权资源）
- 语音：后端 `edge-tts`（微软神经中文音色 Yunjian / Yunxi / Xiaoyi），需后端在线；磁盘缓存 `server/data/tts_cache/`
- **首次点击画面**解锁 AudioContext 与播报
