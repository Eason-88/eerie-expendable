# 阶段 3 验收说明

## 目标

弱联网闭环：本地存档 + 云同步 + 登录 + 远程配置；具备微信小游戏构建路径说明。

## 后端启动

```powershell
cd d:\project\heiying\server
.\venv\Scripts\Activate.ps1
pip install -e ".[dev]"
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

- Health: http://127.0.0.1:8000/health  
- Docs: http://127.0.0.1:8000/docs  
- SQLite：`server/data/eerie.db`

### 主要 API

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/api/v1/auth/dev-login` | 浏览器/调试登录（device_id） |
| POST | `/api/v1/auth/wechat` | 微信 code 登录（无密钥时 mock） |
| GET/PUT | `/api/v1/saves/current` | 云存档读写（Bearer JWT） |
| GET | `/api/v1/progress` | 关卡进度 |
| GET | `/api/v1/config/client` | 远程配置（含 `cdnBaseUrl`） |

### 冲突策略

- 客户端 PUT 携带 `version`
- 若 `version < 服务器 version` → 返回 `conflict: true`，**服务器存档为准**
- 成功写入后 `version + 1`

## 客户端（web-preview）

```powershell
cd d:\project\heiying\client\web-preview
python -m http.server 5173
```

打开 http://127.0.0.1:5173（需后端已启动）

| 检查点 | 何时写入 |
|--------|----------|
| intro | 开局/重开 |
| explore_cleared | 双异象驱散 |
| sniper | 进入狙击段 |
| empty_nest | 抵达空巢 |
| betrayal | 叛变开始 |
| win | 清剿完成 |

HUD「存档」行显示同步状态。清浏览器 localStorage 后用同一逻辑重登可再拉云端（dev-login 按 device_id；清掉 device 键会变成新号）。

**换设备模拟**：复制 `eerie.device.v1` 到另一配置，或固定 device_id 测试。

## 微信小游戏构建（待 Cocos Creator）

1. 安装 Cocos Creator 3.8+，打开/创建 `client/` 工程  
2. 构建设置选择 **微信小游戏**  
3. 开启分包：音频、高模进分包；首包控制  
4. `platform/wechat.ts` 使用 `wx.login` → `POST /api/v1/auth/wechat`  
5. 大资源 URL 使用远程配置 `cdnBaseUrl`（COS/CDN）  
6. 用微信开发者工具导入构建产物，真机预览通关并验证云存档  

> 本机未装 Creator 时，以 web-preview + FastAPI 完成弱联网验收；微信包体步骤按上表在装好编辑器后执行。

## CDN

`.env` / `.env.example` 中 `CDN_BASE_URL` 下发到 `/api/v1/config/client` 的 `cdnBaseUrl`。  
音频与大贴图应上传至该前缀，客户端按配置拼接，不进首包。
