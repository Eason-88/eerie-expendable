## 启动后端

```powershell
cd d:\project\heiying\server
.\venv\Scripts\Activate.ps1
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
```

浏览器打开：http://127.0.0.1:8000/health  
应返回：`{"status":"ok", ...}`

## 启动镜头预览（阶段 0）

用任意静态服务器打开 `client/web-preview/index.html`，例如：

```powershell
cd d:\project\heiying\client\web-preview
python -m http.server 5173
```

浏览器打开：http://127.0.0.1:5173  
**WASD** 移动胶囊体；相机身后跟拍 + 雾；HUD 显示 API health。
