# 阶段 1 验收说明

## 目标

可玩的「走、藏、射」；测试关清完一小波敌人后进入 `Win`。

## 已完成

| 任务 | 状态 | 位置 |

## 操作

| 键 | 作用 |
|----|------|
| 点击画面 | 锁定鼠标 |
| 鼠标 | 瞄准 |
| WASD | 移动（掩体内 A/D 探头） |
| 左键 / 空格 | 射击 |
| E | 进入 / 离开掩体 |
| R | 换弹 |
| 再来一波 | 胜利或死后重置 |

## 启动

```powershell
cd d:\project\heiying\client\web-preview
python -m http.server 5173
```

打开 http://127.0.0.1:5173  

**验收**：击倒全部 5 名敌人 → 出现「清剿完成」，状态机为 `Win`。
