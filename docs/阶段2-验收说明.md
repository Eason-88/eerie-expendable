# 阶段 2 验收说明

## 目标

第一关完整三幕叙事闭环（异象 → 狙击空巢 → 叛变清剿）。

## 启动

```powershell
cd d:\project\heiying\client\web-preview
python -m http.server 5173
```

打开 http://127.0.0.1:5173  

阶段 1 战斗场保留在：http://127.0.0.1:5173/arena.html

## 三幕流程

| 幕 | 状态机 | 怎么触发 | 看什么 |
|----|--------|----------|--------|
| A 潜入异象 | `Explore` | 西侧粉色信标（列车）/ 东侧黄色信标（仪式）本就存在 | **猪头怪常驻车顶，开枪才散**；**稻草人常驻，开枪或靠近才散** |

| B 空巢狙击 | `Sniper` | 两处异象都看过（或强行深入）后向北 | 红色狙击预警线；进掩体可免伤；抵达狙击位发现空巢 |
| C 叛变清剿 | `Betrayal` → `Win` | 空巢后约 3.5 秒 | 对讲机宣布黑鹰7号叛变；友军开火；清剿完毕过关 |

## 对讲机规则

- **只能接收**：顶部电台字幕
- **总部催促**：周期性「黑鹰7号请回话」
- **按 T**：尝试发射 → 提示「对讲机损坏 · 只能接收」

## 操作

| 键 | 作用 |
|----|------|
| WASD / 鼠标 | 移动瞄准 |
| 左键 / 空格 | 射击 |
| E | 掩体 |
| T | 尝试回话（失败） |
| R | 换弹 |

## 路径提示

1. 出生点偏南；先找地面上的**粉色 / 黄色信标**  
2. **西边粉色** → 列车顶猪头怪（开枪驱散）；**东边黄色** → 稻草人（开枪或走进圈内驱散）  
3. 两处驱散后向北进狙击巷  
4. 叛变后清掉绿装友军
